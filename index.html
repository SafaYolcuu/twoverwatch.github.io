Overwatch

Kabile savunma verisini haritada stack renkleri, duvar, gelen saldırı ve gözetleme kulesi menzili ile gösterir.

Gereksinim: Kabile lideri/yetkilisi olmalısın. Script yalnızca harita (map) ekranında çalışır.

Domain alanı: GitHub Pages → Custom domain kısmına bir şey yazmana gerek yok. Boş bırak.

1. Bookmarklet — kopyala yapıştır

Tarayıcıda yeni yer imi oluştur. Ad: Overwatch. URL alanına aşağıdaki satırın tamamını yapıştır (tek satır):

javascript:void((function(){try{sessionStorage.setItem('overwatch_autostart','1');window.__OVERWATCH_BOOT_NOW__=1;var s=document.createElement('script');s.src='https://cdn.jsdelivr.net/gh/SafaYolcuu/twoverwatch.github.io@main/overwatch.js?'+Date.now();s.onerror=function(){alert('Overwatch scripti indirilemedi.');};(document.head||document.body).appendChild(s);}catch(e){alert('Bookmarklet hatasi: '+e);}})())

3. Kullanım

  1. Tribal Wars’ta harita ekranına git.
  2. Yer imlerinden Overwatch’a tıkla.
  3. Veri çekilir, panel açılır, harita renklendirilir.
  4. Ayarları değiştirdikten sonra Haritayı yenile’ye bas.
